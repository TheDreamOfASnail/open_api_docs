Mkdocs工具

安装mkdocs

需要先安装Python和pip，再安装mkdocs，参考(http://www.mkdocs.org/)

clone open_api_docs项目

一定要使用SSH的链接clone ( 先在github中设置好sshKey的值 )

$ git clone git@github.com:xsio/open_api_docs.git

mkdocs 命令

本地测试

$ mkdocs serve

建立site文件

本地测试

$ mkdocs build

发布到ConvertLab文档中心

本地测试

$ mkdocs gh-deploy

注意：mkdocs gh-deploy只push了gh-pages分支，master分支需要另外push
