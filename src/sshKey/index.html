生成SSH keys

1. 检查SSH keys是否存在

$ ls -al ~/.ssh

2. 生成SSH keys

$ ssh-keygen -t rsa -b 4096 -C your_email@example.com

-t 的意思是选择key的type。分别有 RSA 和 DSA 两种。-b 指定秘钥长度。 -C 注释内容，一般为邮件地址。

现在私钥被放在了~/.ssh/id_rsa，而公钥被放在了 ~/.ssh/id_rsa.pub 这个文件里。使用命令行或者进入文件使用编辑器打开并复制key，将key输入github里。查看key命令如下

$ cat id_rsa.pub

3. 在github中设置SSH keys

  1. 第一步：选择setting

  2. 第二步：选择ssh and GPG keys

  3. 第三步：点击右上角的new SSH key按钮

  4. 第四步：输入title，输入key值，点击Add SSH key
